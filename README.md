# About
Package that will help you fix element position on scroll

# Installation

```
npm install aaffix
```
or
```
yarn aaffix
```


# Usage

You can install the plugin

```js
import AAffix from 'aaffix';
Vue.use(AAffix)
```

or

use only the component

```js
import AAffix from 'aaffix';
Vue.component(AAffix.name, AAffix
```
